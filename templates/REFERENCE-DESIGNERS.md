## 7 Hesap Analizi

### 1 · fraxbit · @fraxbit · fraxbit.com
- **Ne yapar:** Premium web design + growth agency
- **Tanıtım dili:** "We _design_ it. We _build_ it. We _grow_ it." — italic vurgu, kinetic
- **Görsel imza:** Marble bust / sculpture imagery (imperfect vs perfected metafor)
- **Renk:** Black + white + bronze/gold minimal, high-contrast
- **Tech stack:** Figma, Framer, Webflow, Blender, React, WordPress
- **Animasyon:** **"INTERACTIVE SCROLL ANIMATIONS" + "GSAP / ADVANCED MOTION" + "3D WEBSITE ELEMENTS" + "EXPERIMENTAL WEB INTERACTIONS"**
- **Layout:** 4-case portfolio grid, modular card sections, bold hero
- **Çıkarım:** Kinetic italic typography + GSAP pin-scrub + sculpture imagery — Template 11'in çekirdeği

## 2026 Web Design Trend Raporu (Awwwards + Figma + Envato Elements)

### Kazanan 5 Trend

| # | Trend | Ne Yapıyor | Technik |
|---|-------|------------|---------|
| 1 | **Kinetic Typography** | Statik metinleri dinamik storytelling'e çeviriyor — hareket, uzatma, dönme, scroll/cursor reactive | Variable fonts (opsz, wght, slnt axis), SplitText, GSAP, WAAPI |
| 2 | **Bento Grid 2.0** | Trendy novelty'den standart'a — %67 top SaaS site. Artık asymmetric, kart-benzeri bloklar, rhythm + movement | CSS Grid + asymmetric span |
| 3 | **Maximalism** | Rich colors, overlapping visuals, bold fonts, dense composition. Spotify + Liquid Death örnek | Layered backgrounds, overflow visual, mixed fonts |
| 4 | **3D / Spline / R3F** | Spline ve React Three Fiber ile dedicated 3D dev gerekmez. Lightweight, mobile-hızlı | Spline embed, R3F + drei, Three.js procedural |
| 5 | **Scroll-Triggered + Real-Time 3D** | Locomotive + Active Theory tarzı immersive. Scroll-linked kamera, dynamic effect | GSAP ScrollTrigger + R3F, Lenis + scroll-linked anim |
